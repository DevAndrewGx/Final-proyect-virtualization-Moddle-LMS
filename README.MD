# Objetivo 

El proyecto final tiene como objetivo que los estudiantes apliquen de manera práctica los conocimientos adquiridos en la gestión de servidores, virtualización, automatización y monitoreo de servicios.

Cada equipo deberá seleccionar un servicio en línea ya existente y funcional (por ejemplo, una plataforma de gestión de aprendizaje – LMS, un sistema de blogs, un CRM, entre otros). Dicho servicio deberá ser implementado en un entorno virtualizado utilizando contenedores (Docker o Kubernetes).

## Plataforma seleccionada

Moddle LMS 

## Justificación 

Se seleccionó Moodle por ser una plataforma ampliamente utilizada en entornos educativos, con arquitectura cliente-servidor, soporte para contenedores Docker y facilidad de integración con herramientas de monitoreo.
